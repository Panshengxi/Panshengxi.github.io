项目：量格商城

http://www.359mai.com/

开发技术： HTML5 + CSS + JQuery + Javascript + Ajax

项目描述：

此项目是一个电商网站，主要功能包括：商品搜索、商品展示、商品评论、购物车、下单流程、在线客服、在线支付、订单状态查询; 管理员后台管理功能包括：商品管理、订单管理、用户管理、客服管理、权限管理、业务分析与统计。

项目职责：我主要负责网站的首页和商品展示、商品搜索、购物车功能的实现、下单管理页面的编写和js的动态效果展示。

工作内容：

1、使用HTML5的语义化标签进行布局，对输入框使用H5的新特性，便于SEO。使用CSS简单的对样式的修饰，实现简单网页展示。

2、使用JQuery配合JavaScript实现轮播、公告滑动等复杂的逻辑代码。

3、使用Ajax异步数据请求调用后端接口，由后端返回数据，解析数据、编译模板渲染HTML页面中，实现动态加载商品列表。

4、使用JavaScript和JQuery完成订单管理，购物车等一系列的时时数据的操作。

项目：广州实验中学管理系统

开发技术： JavaScript + HTML5+ CSS/CSS3+ jQuery +Ajax +JQuery.cookie+Bootstrap+Echarts+requreJS+artTemplate

项目描述：

项目职责：我主要负责学生注册个人信息、学生信息展示、添加，删除，编辑学生信息的页面编写和js效果展示。

工作内容：

1、根据项目需求，使用bootstrap进行页面的排版布局。

2、使用Ajax异步请求调用后端提供的接口，获取后端返回的数据，解析数据、编译模板渲染HTML页面中，实现动态加载学生列表。

3、使用ajax进行验证登录、前后台的数据交互。

4、使用JavaScript配合JQuery获取用户选择需要操作的对象，进行遍历，取值，修改等一系列的逻辑操作。

5、使用requirejs的模块化思想对后台增、删、改、查操作的页面编写。
